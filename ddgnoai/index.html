DuckDuckGo (no AI)

Use this page to add a DuckDuckGo search entry that avoids the AI summaries. The accompanying opensearch.xml sits in the same folder.

  1. Open this page with your web browser.
  2. When prompted, add "DDG no AI" as a custom search engine.
  3. Select it as your default search if you prefer.

If your browser doesn't auto-detect OpenSearch, you can manually import the OpenSearch description from the same directory.
